kQuery QRCode

KRL jQuery QRCode Module

The QRCode module is a port of the jquery.qrcode.js jQuery plugin which provides a pure browser qrcode generation. The module enables you to easily add qrcodes to any webpage.

Download Latest version from GitHub

Using the Module

use module a169x522 alias QRCode
...
rule init_qrcode {
  select when pageview ".*"
    QRCode:init();
    QRCode:qrcode("#qrcode",
      "http://edorcutt.github.com/");
}

Add to Ruleset

Include the use module statement in the meta section of your ruleset. Then within a rule call the init() action, then the qrcode() action within the module. This is a public Kynetx Module, so it is available for use by all KRL developers.
